# uet-agritech-pilot

All the apps related to the pilot phase of UET-Agritech project are maintained in this repo. The following apps are being housed here:

    1. Supervisor - Pilot Phase Questionnaire Data Collection App
    2. Farmer - Machinery Booking App
    3. Owner App No.1 - Machinery Booking Management and Billing Information App
    4. Owner App No.2 - Sensor and Location Data Collection and Dispatch App
    5. Web Dashboard

The UI/UX requirements for the apps are given in the wiki.
